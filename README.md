# randemon
Generate a random Pokémon team.

## About
Randemon is a complete web-app that let's you generate random Pokémon team.

You can try it [here](https://randemon.vercel.app/).

The project is based on [Node.js](https://nodejs.org/en/), [GraphQL](https://graphql.org/), [Redis](https://redis.io/), [React](https://reactjs.org/) and [PokéAPI](https://pokeapi.co/).

## Usage
This repo is a monorepo, navigate to the corresponding folder to get more informations on how to start each project.
